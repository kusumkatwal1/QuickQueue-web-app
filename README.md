# QuickQueue-web-app
QuickQueue is a lightweight web-based appointment and waitlist management system designed for small businesses such as salons, barbershops, and campus service offices.

The system helps staff manage walk-in customers, schedule appointments, update service status in real time, and maintain organized customer records through a simple and user-friendly interface.

## Features

- Customer waitlist management
- Appointment scheduling
- Real-time status updates
- Dashboard overview
- CRUD operations
- SQLite database integration
- Responsive and simple UI

## Technologies Used
  - Python
  - Flask
  - SQLite
  - HTML
  - CSS
