# QuickQueue-web-app
QuickQueue is a lightweight web-based appointment and waitlist management system designed for small businesses such as salons, barbershops, and campus service offices.

The system helps staff manage walk-in customers, schedule appointments, update service status in real time, and maintain organized customer records through a simple and user-friendly interface.

## Features

- Customer waitlist management
- Appointment scheduling
- Real-time status updates
- Dashboard overview
- CRUD operations
- SQLite database integration
- Responsive and simple UI

## Technologies Used
  - Python
  - Flask
  - SQLite
  - HTML
  - CSS
    
### Development Tools
  - VS Code
  - Git & GitHub
    
### Waitlist Management
  Staff can:
  - Add customers to a live waitlist
  - Update service status
  - Remove completed entries
  - Track queue progress

### Appointment Scheduling
  The system allows:
  - Appointment booking
  - Viewing upcoming appointments
  - Updating appointment status
  - Managing scheduling records

### Dashboard
The dashboard provides:
  - Waitlist summary
  - Appointment overview
  - Navigation between modules
  - Live database-driven statistics
